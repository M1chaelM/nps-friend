\question
Consider the following formula: $A= q \land (p \lor \lnot p)$.

\begin{parts}


\part[2] Which of the following correctly characterize(s) $A$? (Select all that apply.)

$\square$ Valid \hspace{1cm} $\square$ Falsifiable \hspace{1cm} $\square$ Satisfiable \hspace{1cm} $\square$ Unsatisfiable

\vspace{.5cm}

\part[2] Which of the following correctly characterize(s) $\lnot A$? (Select all that apply.)

$\square$ Valid \hspace{1cm} $\square$ Falsifiable \hspace{1cm} $\square$ Satisfiable \hspace{1cm} $\square$ Unsatisfiable
\end{parts}
\vspace{8mm}	



\question
Let $A$ be the formula $(p \implies q) \implies  \lnot (p \land q)$
\begin{parts}
	
\part[3] Write $A$ in conjunctive normal form.

\vspace{7cm}

\part[2] Show $A$ is valid or give an interpretation that falsifies $A$. 
\end{parts}
\vspace{5cm}

\question[2]
Let $U$ be a set of formulas, and let $A$ be any formula.\\
If $U$ is unsatisfiable, what can you say about $U\cup A$?
